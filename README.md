The data in the file simulated_assessment_300_students.csv was produced by Microsoft Co-Pilot after the prompt 'Simulate a 100 mark
total mathematical assessment completed by 300 students with 10 questions
of different lengths and difficulty. Whereby the observed score X = T + E
such that T ∼ N (55, 172) and E ∼ N (0, 92), ensure each individual mark
per question per student is displayed, is an integer and that the data is exportable'.
